# Optional Skills

This directory stores non-core skills that are kept in the repository for evaluation,
promotion, or later packaging, but are not part of the default bundled CrawClaw skill set.

These skills are not discovered by the default bundled skill loader because the loader only
scans `<packageRoot>/skills`.

Use this directory for:

- vertical or domain-specific skills
- channel-specific suites
- experimental skills
- internal maintainer workflows
- promotion candidates that are not yet part of core

When a skill graduates into the default product surface, move it into `../skills/`.
