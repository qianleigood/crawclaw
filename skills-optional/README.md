# Optional Skills Catalog

This directory is an optional skill catalog.

It is not part of the main runtime code path and should not be read as product
core architecture. These skills are content/capability assets that may be
installed, referenced, or adapted later.

Use this directory for:

- optional skills that are not part of the default shipped set
- experiments in skill packaging
- skill bundles that are valuable to keep in-repo without treating them as runtime code

If the repository is reorganized later, this directory should likely move under
an explicit catalog-style parent rather than stay at the repo root.
