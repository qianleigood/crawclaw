# AI Pattern Catalog

Use this reference when you need the detailed pattern list while editing text.

## High-frequency pattern groups

### Content inflation
- exaggerated significance / legacy claims
- overblown importance framing
- vague “broader trend” statements
- formulaic challenges / future outlook sections

### Language fingerprints
- overused AI vocabulary
- unnecessary avoidance of simple forms
- negative parallelism
- overused rule-of-three structure
- synonym cycling
- fake scope inflation

### Style fingerprints
- excessive em dashes
- too much bold text
- inline heading + vertical list habits
- over-regular title formatting
- awkward emoji insertion
- unnatural quotation styling

### Interaction fingerprints
- collaboration meta-talk
- knowledge cutoff disclaimers
- ingratiating or overly flattering tone

### Filler / evasion
- filler phrases
- excessive hedging
- generic positive conclusions

## How to use this catalog

1. identify the dominant pattern group
2. rewrite only the affected part first
3. preserve meaning
4. re-check rhythm and tone after cleanup
