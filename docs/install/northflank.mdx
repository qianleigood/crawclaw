Deploy CrawClaw on Northflank with a one-click template and connect to it with a supported gateway client.
This is the easiest "no terminal on the server" path: Northflank runs the Gateway for you.

## How to get started

1. Click [Deploy CrawClaw](https://northflank.com/stacks/deploy-crawclaw) to open the template.
2. Create an [account on Northflank](https://app.northflank.com/signup) if you don't already have one.
3. Click **Deploy CrawClaw now**.
4. Set the required environment variable: `CRAWCLAW_GATEWAY_TOKEN` (use a strong random value).
5. Click **Deploy stack** to build and run the CrawClaw template.
6. Wait for the deployment to complete, then click **View resources**.
7. Open the CrawClaw service.
8. Connect a supported gateway client to the public CrawClaw URL and authenticate using your `CRAWCLAW_GATEWAY_TOKEN`.

## What you get

- Hosted CrawClaw Gateway
- Persistent storage via Northflank Volume (`/data`) so config/credentials/workspace survive redeploys
