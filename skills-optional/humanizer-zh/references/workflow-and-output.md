# Humanizer Workflow and Output

Use this reference when you need the operational checklist and delivery format.

## Core workflow

1. identify AI-like patterns
2. decide the target tone
3. rewrite the problematic sections
4. preserve meaning and factual content
5. do a final pass for rhythm, naturalness, and overcorrection

## Five quick rules

- delete filler
- break formulaic structure
- vary pacing
- trust the reader
- rewrite anything that sounds like a quote-meme or slogan

## Output format

When useful, provide:
- revised text
- short note on what changed
- optional bullets on the strongest remaining risks

## Quality scoring dimensions

Evaluate the rewrite by:
- naturalness
- clarity
- meaning preservation
- tone fit
- remaining AI traces
