# iOS Application Development Guide

Use this skill for native iOS product and UI work.

## Use this skill for

- UIKit or SwiftUI layouts
- iPhone interface design and review
- navigation patterns
- accessibility and Dynamic Type
- dark mode and semantic colors
- iOS-specific system integration

## Working rules

- Keep touch targets >= 44pt.
- Prefer Apple platform conventions over custom navigation metaphors.
- Request permissions in context, not at launch.
- Treat accessibility, Dynamic Type, and safe-area behavior as first-class requirements.
